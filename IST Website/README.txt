##################################################

* The project can be started from index.html file.
* The project need to be run on a server, since it makes many API calls to another server.
* cookies.js file is used to create and read cookies.
* DataTables, js folders are for loading jQuery plugins
* Images folder stores all the pictures for this website.
* proxy.php is for sending and getting back Ajax requests.

##################################################

Description:

* Personalized Voice welcome is given to the user after the first time he/ she enters the name. The name gets stored in Cookies.
* The website does AJAX calls and load contents only when needed. It doesn't waste data by loading the entire website all at once.
* In 'UG Minors' section each course can be chosen to view more details about that course.
* In 'Our people' section, faculty/ staff need to be clicked to view their details.
* Section 'Area of interest' has a wiggly effect on it's div boxes.
* The student resources section is given a feature of 'eye movement', which responds to mouse cursor movement.
